# About

Completionistv2 is a web site created for people who aim for osu!completionist or just want to track their overall progress in osu!. It supports all the gamemodes alongside all the variants (gamemode specific, converts, both). It aims to support you through your journey with a lot of additional features.

Some of the features:

- Check your progress per year in a clean way
- Browse through and check individual maps
- Use filters to find exactly what you're looking for
- Export your progress with built-in tools
- Generate collections that'll aid you ingame

Check out the site and https://lziobro.github.io/completionistv2.
